# Nomadcare Official Website

This repository contains the **official website** for the **Nomadcare** project, developed during the [FORGe AI Hackathon](https://lisbonaiweek.com/forge-ai-hackathon) at Lisbon AI Week 2025.

## About Nomadcare

Nomadcare is an innovative B2B telemedicine solution focused on providing language interpretation services for healthcare, targeting both the US and Portugal markets. The platform addresses real challenges faced by international patients in medical settings, aiming to bridge communication gaps with multimodal features like gesture recognition and video call support.

## Hackathon Context

- Developed in Lisbon at the AI Hub Penha de França, during Portugal’s largest student AI hackathon.
- Organized by NOVAe (NOVA IMS Entrepreneurship Club) and supported by industry partners such as Daredata, NOVA IMS ElevenLabs, Beyond Presence, N8n, and Data Makers Fest.
- Over 100 students in AI, machine learning, data science, and software engineering collaborated on hands-on prototyping and pitching.

## Features

- Official landing page presenting Nomadcare’s concept and core value proposition.
- Focus on medical interpreter assistant solutions for global healthcare.
- Modern tech stack (TypeScript, React/Next.js).
- Storytelling and pitch-ready assets for investors and healthcare partners.
